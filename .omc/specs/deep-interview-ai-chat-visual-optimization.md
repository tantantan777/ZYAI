## 目标
将AI对话页面 (frontend/src/pages/AIChat.tsx 和 AIChat.css) 优化为专业商务风格，通过综合运用边框、背景色、阴影等视觉手段，增强区域分割和层次感，使界面符合建筑项目管理平台的专业定位。

## 约束条件
- 必须保持现有功能完整性，不改变交互逻辑
- 需要与系统其他页面（Dashboard、项目管理、用户权限等）的视觉风格保持一致
- 优化仅限于CSS样式调整，不涉及组件结构重构
- 需要保持响应式布局，移动端适配不能受影响
- 配色方案需符合专业商务定位（避免过于鲜艳的颜色）

## 非目标
- 不添加新功能或改变现有功能
- 不修改组件的交互行为
- 不改变页面布局结构（左侧历史列表 + 右侧聊天区的整体布局保持不变）
- 不涉及性能优化或代码重构

## 验收标准
- [ ] **区域边界清晰明确**：用户能快速识别历史列表、聊天区、输入框等不同功能区域，各区域之间有明显的视觉分隔
- [ ] **专业商务配色统一**：整体配色协调统一，符合建筑项目管理平台的专业定位，与系统其他页面风格一致
- [ ] **信息层次分明**：主要内容（聊天消息）突出，次要信息（时间戳、元数据）弱化，用户注意力自然聚焦在核心内容上
- [ ] **视觉层次丰富**：通过边框、背景色、阴影等多种手段综合运用，形成清晰的视觉层次
- [ ] **响应式适配正常**：移动端和桌面端都能正常显示，视觉优化在不同屏幕尺寸下都有效

## 具体优化方案

### 1. 边框优化
- 主要分隔线从 1px 增加到 2px
- 边框颜色从 #f0f0f0 调整为 #e0e0e0（更明显）
- 关键区域（左右分隔、顶部、底部）使用更粗的边框

### 2. 背景色优化
- 左侧历史列表：从 #fafafa 调整为 #f5f5f5（更明显的灰）
- 右侧聊天区消息列表背景：保持 #fafafa
- 顶部标题栏和底部输入区：使用纯白 #ffffff
- 历史对话卡片选中状态：从 #e6f4ff 调整为更深的蓝色背景

### 3. 阴影优化
- 左侧历史列表：添加右侧阴影 `box-shadow: 2px 0 8px rgba(0,0,0,0.06)`
- 顶部标题栏：添加底部阴影 `box-shadow: 0 2px 8px rgba(0,0,0,0.06)`
- 底部输入区：添加顶部阴影 `box-shadow: 0 -2px 8px rgba(0,0,0,0.06)`
- 消息气泡：添加轻微阴影 `box-shadow: 0 1px 4px rgba(0,0,0,0.08)`
- 历史对话卡片：添加悬停阴影效果

### 4. 配色方案
- 主题蓝色：#1890ff（保持与系统一致）
- 深灰文字：#262626（主要文字）
- 中灰文字：#8c8c8c（次要文字）
- 浅灰文字：#bfbfbf（辅助信息）
- 边框灰：#e0e0e0（主要边框）
- 背景灰：#f5f5f5（区域背景）

## 实施建议

### 优先级
1. **高优先级**: 边框和背景色优化（快速见效，影响最大）
2. **中优先级**: 阴影效果添加（提升层次感）
3. **低优先级**: 细节调整（圆角、间距微调等）

### 风险提示
- 阴影效果不宜过重，避免显得过于立体或花哨
- 背景色对比度要适中，避免过于刺眼
- 需要在不同屏幕尺寸下测试效果
- 注意与系统其他页面的视觉一致性
